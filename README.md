### Internship Project - Event Management Application - 2020
<br/>

> Back-end: SpringBoot<br/>
> Front-end: React<br/>
> Database: PostgreSQL<br/>

<br/>

Event Management Application that users can perform basic CRUD operations on events.<br/><br/>
Users can use the app based on their roles. Create&join to the events, generate event QR code and can send it via email, etc.<br/><br/>
App provides basic Auth functions in the application.<br/>
